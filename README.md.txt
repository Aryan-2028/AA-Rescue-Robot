🤖 AA Rescue Robot : AI Powered Rescue & Surveillance Robot using Arduino UNO + ESP32-CAM
 🚀 Project Overview
AA Rescue Robot is a smart surveillance and rescue robot designed for learning robotics, embedded systems, sensor integration, and real-time web monitoring.
The robot combines multiple sensors with an ESP32-CAM live monitoring system to create a compact rescue and detection platform.
This project was mainly built to explore:
* Arduino programming
* ESP32-CAM integration
* Sensor communication
* Web dashboard development
* Robotics automation
* Real-time monitoring systems

🔥 Features
* 📷 ESP32-CAM live video streaming
* 🌐 Real-time web dashboard
* 🚨 Obstacle detection
* 🔥 Fire detection
* 🌫️ Gas detection
* 📏 Ultrasonic distance sensing
* 🔊 Buzzer alert system
* ⚡ Fast AJAX live sensor updates
* 📡 ESP32 hotspot support
* 🎮 Motor control system
* 🛠️ Expandable hardware architecture

🧠 Technologies Used

 Hardware

* Arduino UNO
* ESP32-CAM
* Ultrasonic Sensor
* Gas Sensor
* Temp Sensor (DHT 11)
* Motor Driver
* DC Motors
* Buzzer

 Software
